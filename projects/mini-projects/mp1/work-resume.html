Work Experiences

Frontend Engineer, Internship - Classum

January 2023 - July 2022

Technologies: TypeScript, React-Native, Figma

As a Frontend Engineer Intern at Classum, I played an integral role in the UI/UX frontend development of an app designed as a private open communication-based community tool for organizations such as companies and schools. The application is now actively serving over 11,000 clients across more than 32 countries.

My contributions to the project included:

  • Utilizing Metro as the bundler for app development to achieve faster bundling times, efficient optimization, and seamless integration with React Native, which provided distinct advantages over webpack.
  • Implementing a robust frontend-backend communication framework by integrating Axios for REST API calls, optimizing asynchronous operations, and data transfer using Data Transfer Objects (DTOs).
  • Employing Yarn for package management and version control alongside Fastlane to automate various development workflow tasks, ensuring efficient testing, deployment, and release processes.
  • Enhancing code quality and maintainability by implementing ESLint and leveraging TypeScript's static type checking within React Native development to detect errors early.
  • Optimizing application performance, especially in the rendering of substantial data, by leveraging Recoil for state management.
  • Creating smooth and responsive animations using React Native Reanimated, prioritizing app performance and user experience.
  • Integrating Bugsnag for real-time error tracking and Flipper for debugging purposes, which included network checks and performance monitoring.
  • Developing a notifications feature using deep linking and react-native navigation to enhance user engagement within the app.
